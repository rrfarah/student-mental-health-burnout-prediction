# Student Mental Health and Burnout Prediction

Predictive analysis of student mental health and burnout using machine learning techniques and explainable AI methods.

---

## Overview

This project focuses on analyzing and predicting student burnout levels based on mental health indicators, academic pressure, lifestyle factors, and behavioral patterns using machine learning algorithms.

The project also applies Explainable AI (XAI) techniques such as SHAP and LIME to improve model interpretability and understand feature contributions in burnout prediction.

---

## Objectives

- Analyze factors contributing to student burnout
- Identify relationships between mental health indicators and academic performance
- Build predictive machine learning models
- Evaluate model performance using classification metrics
- Interpret predictions using Explainable AI techniques

---

## Dataset

The dataset used in this project contains student mental health and burnout-related attributes such as:

- Stress level
- Sleep duration
- Academic pressure
- Anxiety indicators
- Depression symptoms
- Lifestyle habits
- Burnout level

Original dataset source:

```text
https://www.kaggle.com/datasets/sehaj1104/student-mental-health-and-burnout-dataset
```
