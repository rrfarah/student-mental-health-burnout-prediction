# Student Mental Health and Burnout Prediction

Predictive analysis of student mental health and burnout using machine learning techniques.

---

## Overview

This project aims to analyze and predict student burnout levels based on mental health indicators, academic factors, and behavioral patterns using machine learning models.

The notebook includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature engineering
- Model training
- Model evaluation
- Burnout prediction analysis

---

## Objectives

- Identify factors contributing to student burnout
- Analyze mental health patterns among students
- Build predictive machine learning models
- Evaluate model performance for burnout prediction
